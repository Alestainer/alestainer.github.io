Verification Process Documentation

When users register for our service:

  1. Users complete the registration form above
  2. The SMS consent checkbox is unchecked by default
  3. Users can complete registration regardless of whether they opt in to SMS
  4. Opted-in users receive a confirmation message with opt-out instructions
  5. All message templates include opt-out instructions
